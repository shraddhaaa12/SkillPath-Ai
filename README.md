# SkillPath AI

A simple React-based educational website developed as part of the AI Driven Web & Product Development assignment.

## Features

- Home (Landing) Page
- About Page
- Contact Page
- 404 Not Found Page
- Navigation using React Router
- Responsive layout
- Contact form with validation
- Success and error messages using React useState

## Tech Stack

- React.js
- React Router DOM
- JavaScript (ES6+)
- CSS3
- Vite

## Installation

Clone the repository

```bash
git clone https://github.com/shraddhaaa12/SkillPath-Ai.git
```

Go to project folder

```bash
cd SkillPath-Ai/frontend
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
